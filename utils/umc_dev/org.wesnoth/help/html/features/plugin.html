The plugin has the following list of abilities (not exhaustive):

  • Easily creation of UMC Addons
  • Quick "Writing" of WML code by using WML Wizards, by focusing on the content
  • Integrated launch of the Game or Map Editor
  • Integrated launch of the WML tools on individual projects / files
  • Support for multiple Wesnoth versions
  • Integrated powerful WML Editor with autocomplete, go to macro definition, etc
